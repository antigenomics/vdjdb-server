VDJdb: TCR specificity database web browser
===========================================

The `web application <https://vdjdb.cdr3.net>`__ can be used to browse and query the VDJdb database, providing an intuitive interface for database navigation.
Additionally, it can be used to query immune repertoire sequencing (RepSeq) samples against the database and generate sample-level summary of detected antigen specificities.
This help page describes the installation and setup of VDJdb server at your local network, as well as the REST API for the vdjdb.cdr3.net portal.

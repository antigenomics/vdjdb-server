About

VDJdb web browser is developed and maintained by Dmitry Bagaev (@@bvdmitri). The software is open-source and free, the source code is hosted Visit W3Schools.com!. VDJdb-server can be also installed locally following the instructions provided here

Credits

VDJdb web browser application (VDJdb-server) is developed and maintained by Dmitry Bagaev (@@bvdmitri).

Standalone API for VDJdb and database management system (VDJdb-standalone and VDJdb-db) is developed and maintained by Mikhail Shugay (@@mikessh).

VDJdb database is a collaborative effort within the framework of Advanced bioinformatics platform for PERsonalized cancer IMmunotherapy (APERIM) project. APERIM is a Horizon2020-funded project coordinated by the Medical University of Innsbruck, Austria. For more information please see aperim.eu.

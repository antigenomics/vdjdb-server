The software

VDJdb web browser application is an open-source software implemented using Play Framework and Bootstrap. A standalone browser version of the software can be installed locally, the following link contains the software package and installation instructions.

Users interested in contributing TCR sequences of known antigen specificity to the database or wishing to obtain the most recent database version as a plain-text table can do so by following this link.

A standalone command-line software (VDJdb-standalone) should be preferred in cases when huge amounts of immune repertoire sequencing data are being analyzed or when users are interested in obtaining unprocessed output for post-analysis purposes. The corresponding software package is available here.

You might be also interested in our lightweight immune repertoire browser VDJviz

VDJdb live support is provided via Gitter

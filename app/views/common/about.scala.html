The software

VDJdb web browser application is an open-source software implemented using Play Framework and Bootstrap. A standalone version of the software can be installed locally, the following link contains the software package and installation instructions.

Users interested in contributing TCR sequences of known antigen specificity to the database or obtain the database as a plain-text table can do so by following this link.

A standalone command-line software (VDJdb-standalone) should be preferred in cases when huge amounts of immune repertoire sequencing data are being analyzed or when users are interested in obtaining unprocessed output for post-analysis purposes. The corresponding software package is available here.

You might be also interested in our lightweight immune repertoire browser VDJviz

Credits

VDJdb database is a collaborative effort within the framework of Advanced bioinformatics platform for PERsonalized cancer IMmunotherapy (APERIM) project. APERIM is a Horizon2020-funded project, for more information please see aperim.eu.

VDJdb web browser application (VDJdb-server) is developed and maintained by Dmitry Bagaev (@@bvdmitri).

Standalone API for VDJdb and database management system (VDJdb-standalone and VDJdb-db) is developed and maintained by Mikhail Shugay (@@mikessh).

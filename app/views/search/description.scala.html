Database browser

Press SEARCH to display the table of database records.

Database search can be narrowed by adding text and/or sequence filter in panels below:

  • Exact (with autocomplete), substring and REGEX pattern matching filters can be applied to all database columns.
  • Database records can be filtered by VDJdb confidence score value.
  • Filtering by global alignment to pre-defined amino acid sequence with a fixed number of allowed mismatches is available for CDR3 and antigen sequence columns.
